◆ Cdimension()

void GameObjectlib.CameraContainer.Cdimension ( )

　orthographicの真偽をStatusと対応させたまま変更するメソッド

先にStatusの値を変えその値を元にorthographicを変更する
これによりStatusとorthographicの対応に齟齬が起こってもこのメソッドを呼び出すことで直る

覚え書き
Statusの値の変更は１と最下位ビットのXOR演算によるもの
Status=2のとき：10(二進数)と1(二進数)一桁目のXORは0と1になる。そして0≠1であるから一桁目が１になり11(3(十進数))になる
Status=3のとき：11(二進数)と1(二進数)一桁目のXORは1と1になる。そして1=1であるから一桁目が１になり10(2(十進数))になる
このことからStatusの値は2と3以外の値に変化しない。
